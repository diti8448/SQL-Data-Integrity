# SQL Data Integrity & Unit Testing (HrvatskaDB)

Ovaj projekt pokazuje kako povezati Python i SQL Server te koristiti testove za provjeru podataka.

## Što sam naučio/napravio:
- **Python & SQL**: Spajanje preko `pyodbc` drivera.
- **Unit Testing**: Korištenje `unittest` modula za automatsku provjeru baze.
- **Unicode**: Ispravan rad s hrvatskim znakovima (č, ć, ž, š, đ).
- **Git**: Osnove verziranja koda.

## Kako pokrenuti:
1. Instaliraj biblioteku: `pip install pyodbc`
2. Pokreni skriptu: `python test_database.py`
